Reads a file within allowed directories.
- `file_path` can be relative to the allowed directories or an absolute path inside the allowed directories.
- Optional: `offset` (1-indexed, default `1`) and `limit` (default `2000`); both must be >= `1`.
- Output is text with line numbers as `L{n}: ...`. Lines longer than `2000` chars are truncated.
- This tool reads files, not directories. Use `glob` to find files or `bash` only when you specifically need a directory listing.
- Paths outside the allowed directories are rejected.
- Missing files return an error. Non-text files are returned as text bytes; there is no special image rendering.
- Read the file before editing it. Read multiple related files in parallel when useful.
