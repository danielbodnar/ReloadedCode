Writes a file within allowed directories and creates parent directories if needed.
- `file_path` can be relative to the allowed directories or an absolute path inside the allowed directories. `content` is required.
- Existing files are overwritten.
- Paths outside the allowed directories are rejected.
- Prefer `edit` for targeted changes.
- If the file already exists, read it first before overwriting.
- Use this for new files or full rewrites, not small edits.
